This Unity VR project creates an immersive 3D escape game where the player navigate a locked room using joystick-based smooth locomotion to reach the final room.

Advanced interactions feature a rotating portal cast system and a fireball projectile mechanic for destroying obstacles, combining spatial magic with physical object manipulation.​

Project Initialization
Requirements
Unity 2022.3 LTS or newer

VR headset with joystick support (Quest, Vive, Index)

XR Interaction Toolkit 2.5+ and OpenXR Plugin​

Setup Instructions
Clone repository: git clone https://github.com/Ariazoox/Escape-Game-Unity-VR.git

Open in Unity Hub with 2022.3 LTS

Package Manager → Install XR Interaction Toolkit \& OpenXR

Project Settings → XR Plug-in Management → Enable OpenXR for your platform

Open main scene: Assets/Scenes/EscapeGameScene.unity

Connect VR headset → Play to test joystick movement and portal/fireball casting​

Controls
Joystick: Smooth continuous movement

Portal Cast: Right and Left trigger + aim to cast rotating spell on the portal

Fireball: Right trigger moving upside down to launch fireball projectile​
